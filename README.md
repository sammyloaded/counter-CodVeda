## 🚀 Project Overview
A minimalistic counter application built to demonstrate the power of Vanilla JavaScript DOM manipulation. This app allows users to interact with the UI in real-time, updating the state of the application without page refreshes.

## ✅ Objectives Accomplished
- **Dynamic DOM Updates:** Uses `textContent` and JavaScript variables to manage app state.
- **Event Handling:** Implemented `click` event listeners for all control buttons.
- **Logic Constraints:** Ensures the counter stays within logical bounds (cannot go below zero).
- **Minimalist Design:** Clean, modern UI with subtle animations and feedback for "out of bounds" actions.
- **UX Feedback:** Added a shake animation and color change when attempting to decrement at zero.

## 🛠️ Tech Stack
- **HTML5:** Clean semantic structure.
- **CSS3:** Flexbox, custom animations, and modern typography.
- **JavaScript:** State management and DOM element selection.

## 🔧 Installation
1. Clone the repository.
2. Open `index.html` in your browser.
3. Use the **+**, **-**, and **Reset** buttons to interact with the counter.
